# tonytnt.github.io

Personal homepage of Chen Chen (陈晨) — https://tonytnt.github.io

A self-contained bilingual (EN/中文) static page: all content, styles and scripts live in `index.html`. No build step — GitHub Pages serves it directly (`.nojekyll`).

- `index.html` — the whole site (edit content here)
- `images/` — profile photo, favicons, publication figures
- `files/pdf/` — CV and paper PDFs

Local preview:

```bash
python3 -m http.server 4000
# open http://localhost:4000
```
